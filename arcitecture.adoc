= {project-name} architecture

== What this is

A group of services for monitoring:

* Battery levels
* Disk space

Users are notified by desktop notifications when thresholds are crossed.

== Architecture overview

=== Approach

A monorepo of rust-based systemd services, for code sharing.

=== Key components

==== Battery monitor service
Monitor battery levels and uses the notification library to alert users. It runs as a systemd service.

==== Disk space monitor service
Monitor disk space usage and uses the notification library to alert users. It runs as a systemd service.

==== Notification library
Shared library for sending desktop notifications. Used by both monitor services.

=== How they connect
The monitor services use the notification library to send alerts. Each service runs independently as a systemd service.
