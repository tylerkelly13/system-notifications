= system-notifier

A Rust-based Debian package

== Building

Build the project:

[source,bash]
----
cargo build --release
----

== Testing

Run tests:

[source,bash]
----
cargo test
----

== Building Debian Package

Build the Debian package:

[source,bash]
----
dpkg-buildpackage -us -uc -b
----

The generated `.deb` file will be in the parent directory.

== Installing

Install the package:

[source,bash]
----
sudo dpkg -i ../system-notifier_*.deb
----
